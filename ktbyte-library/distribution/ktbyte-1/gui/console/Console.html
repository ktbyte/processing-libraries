Name

Console
Description This GUI element acts as a Command-line interface, accepting user input and handling custom responses.

Functionality:


  • writing inside the console's textbox (the console must be focused)
  • deleting entered text using the BACKSPACE key
  • storing user inputs
  • retriving stored entries
  • text size and color customization
  • trigger custom action for each input
Constructor Console(PApplet pap, int x, int y, int width, int height).
This constructs a new Console object within the current context (PApplet), starting from the x and y coordinates, having a given width and height.
