Educational Web-Enabled Library

A Library by KTByte for both the Processing and KTByte coder programming environments.
Last update, 04/26/2018.

A collection of components to help students build interactive sketches.

Download

Download Educational Web-Enabled Library version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted ktbyte folder into the libraries folder of your Processing sketches. Reference and examples are included in the ktbyte folder.

Keywords. Educational

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.

Source. The source code of Educational Web-Enabled Library is available at GitHub, and its repository can be browsed here.
