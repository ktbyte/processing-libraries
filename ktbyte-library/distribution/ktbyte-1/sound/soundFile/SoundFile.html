Name

SoundFile
Description This element can be used to play mp3 files. It is just a wrapper library over the minim library.

Functionality:


  • playing mp3 sounds from a given URL/path
  • play/stop/loop actions
Constructor SoundFile(PApplet parent, java.lang.String path).
This constructs a SoundFile object within the current context (PApplet), pointing to the given path.
Methods isLoaded()
load()
