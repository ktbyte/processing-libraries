Name

playChord()
Examples 
SoundCipher sc;
float[] pitches = {50, 60, 70};

void setup() {
    sc = new SoundCipher(this);
    sc.playChord(pitches, 100, 2.0);
}
Description Plays a chord for the given pitches and sound configuration
Syntax
playNote(pitches, dynamic, duration)
Parameters
pitches float[]: The MIDI pitches (frequencies) for each note [0-127].
dynamic double: The loudness, MIDI velocity, of the note. Should be a value between 0 and 127.
duration double: The length that the note will sound, in beats. Should be a value between 0 and 127.
Returnsvoid
